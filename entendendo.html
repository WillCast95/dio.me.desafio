ENTENENDO A DIFERENÇA ENTRE LISTAS ORDENADAS E NÃO ORDENADAS

A principal diferença entre listas ordenadas e não ordenadas é a forma como os itens são apresentados:

Listas ordenadas:

Também conhecidas como listas numeradas, os itens são enumerados, ou seja, mostrados com uma contagem que os precede. São usadas quando a ordem dos itens é importante.

Listas não ordenadas

Também conhecidas como listas com marcadores, os itens são marcados com bullets, geralmente pequenos círculos pretos. São usadas quando a ordem dos itens não é importante.
